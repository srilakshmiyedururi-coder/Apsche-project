# apsche_ai

A Flask web application that predicts the Human Development Index (HDI) using development-related indicators such as life expectancy, years of schooling, and GNI per capita.

## Project overview
This project includes:
- a trained machine learning model for HDI prediction
- a Flask web interface for entering input values
- a documentation structure prepared for internship submission

## Project folder structure
- Source Code/ - application source files
- Dataset/ - training dataset files
- Models/ - trained model artifacts
- Templates/ - HTML templates
- Static/ - static assets such as CSS files
- Project Documentation/ - internship documentation folders and files
- Training/ - notebook and training assets

## Run locally
1. Install dependencies:
   pip install -r "Source Code/requirements.txt"
2. Start the application:
   cd "Source Code"
   python app.py
3. Open the app in your browser:
   http://127.0.0.1:5000
